# study-planner
A task management app for students

## Target Users
Students managing coursework and deadlines.

## Features
- Create tasks with title, due date, and priority
- View all tasks in a list
- Mark tasks as complete
- Filter tasks by subject
- Set priority levels (low / medium / high)

## Tech Stack
- Next.js (React framework)
- Deployed on Vercel

## User Stories
The Resource Manager: As a student, I want to attach lecture slides and PDF readings to specific tasks, so that I have all my study materials ready when I start a revision session.

The Micro-Planner: As a student, I want to break down large exam topics into smaller sub-tasks, so that I can track my progress through a syllabus without feeling overwhelmed.

The Sync Strategist: As a student, I want to synchronize my assignment deadlines with my school calendar, so that I can see how my study sessions fit around my live classes and seminars.

The Focus Guard: As a student, I want to set 'Deep Work' timers for specific tasks, so that I can stay focused on difficult subjects like Financial Accounting without distractions.

The Collaborative Peer: As a student, I want to share a shared task list with my group project members, so that we can assign roles and track who has finished their part of the presentation.
